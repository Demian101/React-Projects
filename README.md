## Git

将一个本地文件夹 和 github 上创建的 repo 建立联系 : 

```
(git remote rm origin)
git remote add origin https://github.com/Sodaoo/ReactProjects.git
git branch -M main
git push -u origin main
```

初次提交使用 -u , 以后提交就

```bash
git add .
git status
git commit -m '..'
git push origin main
```
